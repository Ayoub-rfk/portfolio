Épreuve E6 — Cybersécurité & Administration avancée

Projets axés sur la sécurisation des infrastructures, le déploiement de services et l'analyse réseau.

L'épreuve E6 (Parcours de professionnalisation) évalue les compétences acquises lors des périodes en entreprise et les projets de cybersécurité. Pour l'option SISR, elle couvre la sécurisation des infrastructures, la gestion des incidents et les bonnes pratiques de sécurité informatique.
// security.projects

Projets Sécurité & Déploiement

🌍
Déploiement Serveur Web Apache sur Debian
Installation d'une stack LAMP complète sur Debian : Apache, MySQL, PHP. Configuration du pare-feu UFW, exposition du service via NAT, sécurisation des accès et configuration des hôtes virtuels.
Apache LAMP UFW Firewall NAT Debian
🔍
Simulation d'Incidents Réseau
Scénarios de pannes simulées : défaillance DNS, perte de bail DHCP, perte de connectivité réseau. Analyse des causes avec les outils de diagnostic (ipconfig, netstat, tracert, Wireshark). Rédaction de rapports d'incidents.
Wireshark DNS / DHCP Traceroute Netstat Rapport d'incident
🔒
Sécurisation d'un Réseau Cisco (Port-Security)
Configuration de la sécurité des ports sur des commutateurs Cisco IOS : limitation du nombre d'adresses MAC, détection d'intrusions, modes de violation (protect, restrict, shutdown). Tests de résistance aux attaques de type MAC flooding.
Cisco IOS Port-Security MAC Filtering Packet Tracer
📊
Supervision avec Nagios & Zabbix
Configuration de checks de supervision sur des serveurs Linux et Windows avec Nagios. Mise en place d'alertes sur des seuils (CPU, mémoire, disque). Découverte des fonctionnalités de Zabbix pour la supervision réseau.
Nagios Zabbix SNMP Linux Windows Server
📡
Analyse de Trames avec Wireshark
Capture et analyse de trames réseau en environnement contrôlé : filtrage de flux HTTP, DNS, ARP, identification d'anomalies, reconstitution de sessions TCP et analyse des handshakes.
Wireshark TCP/IP HTTP / DNS / ARP Nmap
🗄️
Active Directory & Stratégies GPO
Configuration d'un domaine Active Directory sous Windows Server 2022 : création d'utilisateurs, d'unités organisationnelles (OU), application de stratégies de groupe (GPO) pour uniformiser les postes de travail.
Active Directory GPO Windows Server 2022 DNS / DHCP
