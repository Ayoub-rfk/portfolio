This is my portfolio where I share my academique project. This is for educational purpose and the goal is to be able to use GLPI that we downloaded before in a server. 
GLPI (Gestionnaire Libre de Parc Informatique) is a web-based application that helps organizations manage their entire IT infrastructure from hardware inventory to user support tickets it is a single unified platform.

This repository contains the user documentation for GLPI, covering both administrator and user perspectives. Each section can be consulted independently, though reading the full documentation is recommended for a complete understanding of the platform.
