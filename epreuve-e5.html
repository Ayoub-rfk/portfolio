Épreuve E5 — Situations Professionnelles

Projets d'infrastructure, de virtualisation et d'administration réseau réalisés en formation et en autonomie.

L'épreuve E5 (Support et mise à disposition de services informatiques) évalue la capacité du candidat à réaliser des situations professionnelles dans le domaine des réseaux et systèmes. Elle repose sur un dossier de situations professionnelles et une présentation orale devant jury.
// projects

Projets réalisés

🖥️
Déploiement VMware ESXi + Monitoring
Installation de VMware ESXi 6.7 sur baremetal, déploiement de Windows Server 2019 avec Active Directory et DHCP, puis mise en place d'une supervision Nagios pour les clients Windows.
VMware ESXi 6.7 Windows Server 2019 Active Directory DHCP Nagios
☁️
Administration OpenStack Multi-Node
Installation manuelle d'un environnement OpenStack en configuration multi-nœuds. Administration des réseaux clients avec Neutron, configuration de VXLAN et OpenvSwitch pour l'isolation des tenants.
OpenStack Newton Neutron OpenvSwitch VXLAN Linux Bridge
⚡
Migration à chaud KVM (Live Migration)
Réalisation d'une migration à chaud de machines virtuelles sous QEMU-KVM avec stockage partagé NAS via NFS. Utilisation de libvirt et virt-manager pour l'orchestration.
KVM / QEMU libvirt virt-manager NFS / NAS Ubuntu 16.04
🌐
Application Web Multi-Tiers
Développement d'une application web de suivi de consommation d'énergie. Architecture multi-tiers avec un front-end HTML/CSS, un back-end PHP et une base de données MySQL.
HTML / CSS PHP MySQL Architecture MVC
🔧
Administration VMware vCenter
Configuration avancée de VMware : vMotion, migration P2V, Distributed Resource Scheduling (DRS), High Availability (HA) et Fault Tolerance (FT). Stockage iSCSI SAN partagé avec Openfiler.
vCenter 6.7 vMotion DRS / HA / FT iSCSI SAN Openfiler
🔗
LAN Entreprise Simulé sous GNS3
Conception et déploiement d'un réseau LAN d'entreprise sous GNS3 : 3 VLANs, routage dynamique OSPF, sécurisation des ports de commutateurs, test de redondance avec Spanning Tree Protocol (STP).
GNS3 Cisco IOS VLAN / Trunk OSPF STP
